Tancred
=======

This project aims at building an autonomous RC car equipped with a camera.

Material
--------
* Conrad Reely RC car
* Intel Realsense D435 RGBD camera
* NVidia Jetson Nano
* Arduino Uno
* BTS 7960 motor driver
* Homemade 3d printer platform

Software
--------

Dependencies
* dlib
* LCM (lightweight communication and marshalling)
* Qt5
* realsense2
